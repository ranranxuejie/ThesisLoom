# ThesisLoom: A Multi-Agent System for Academic Writing

## Abstract
This section briefly introduces ThesisLoom, a state-graph multi-agent AI system based on LLMs (e.g., Gemini-3-pro) designed to automate the process of planning, drafting, reviewing, and refining full-length academic research papers. It resolves the notorious issue of context degradation in LLMs.

## 1. Introduction
The gap between short form text generation and full-length, structured academic papers. The need for specialized architectures (ThesisLoom) leveraging State-Graph architectures (LangGraph).

## 2. Methodology: Multi-Agent State-Graph Architecture
The architecture comprises several main nodes: Planner (for modularizing tasks and sections), Editor (for context-aware drafting based on existing sections), Reviewer (for analytical critique), and Revise (for structured edits). State variables propagate current textual data, existing material, and feedback loops across states until a predefined quality threshold or condition is met.

## 3. Results and Demonstration
Evaluation of generating complex multi-section papers. Metric comparison based on logic connectivity, adherence to research gaps, factual stability, and stylistic consistency (e.g., Nature/Science journal style).
