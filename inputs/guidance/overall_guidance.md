
## I. Overall Structure Patterns: The IMRAD+ Framework

### 1. Standard Macro-Structure

| Section | Proportion | Core Function | Key Content |
| :--- | :--- | :--- | :--- |
| **Title** | N/A | Capture attention, convey core contribution | Keywords, study type, main variable |
| **Abstract** | 5-7% | Standalone summary of entire study | Background, Methods, Results, Conclusion |
| **Introduction** | 10-15% | Establish research gap and objectives | Context, gap, hypothesis, contribution |
| **Methods** | 20-30% | Enable reproducibility | Design, subjects, procedures, analysis |
| **Results** | 20-25% | Present findings objectively | Data, statistics, figures, tables |
| **Discussion** | 25-35% | Interpret and contextualize findings | Meaning, comparison, implications, limitations |
| **Conclusion** | 5-10% | Final synthesis and takeaway | Core finding, impact, future direction |
| **References** | Variable | Support claims with evidence | Prior work, methodologies, theories |

### 2. Structural Flow Principles

- **Funnel Structure (Introduction)**: Broad context → Specific gap → Research question → Study contribution
- **Inverted Funnel (Discussion)**: Specific findings → Interpretation → Broader implications → Field-level impact
- **Parallel Alignment**: Each research question in Introduction receives corresponding results and discussion
- **Logical Progression**: Each section builds upon the previous, creating coherent narrative arc

## II. Module-Specific Writing Logic and Functions

### 1. Abstract Module

| Component | Function | Standard Content | Typical Length |
| :--- | :--- | :--- | :--- |
| **Background** | Establish context and importance | 1-2 sentences on field significance | 20-30 words |
| **Objective** | State research purpose | Clear research question or hypothesis | 15-25 words |
| **Methods** | Describe approach | Study design, sample, key procedures | 30-50 words |
| **Results** | Present key findings | Primary outcomes with statistics | 50-80 words |
| **Conclusion** | State implications | Main takeaway and significance | 20-40 words |

**Positive Patterns:**
- Use structured abstract format (Background, Methods, Results, Conclusion) when required
- Include numerical results with effect sizes and confidence intervals
- End with clear implication statement
- Maintain consistency with main text findings
- Use keywords that enhance discoverability

### 2. Introduction Module

| Paragraph | Function | Content Focus | Transition Strategy |
| :--- | :--- | :--- | :--- |
| **1** | Establish field importance | Broad context, significance | General → Specific |
| **2** | Review relevant literature | Key prior work, consensus | Build on established knowledge |
| **3** | Identify research gap | What is unknown or contested | "However," "Despite," "Limited" |
| **4** | State research objective | Study purpose, hypothesis | "Therefore," "We aimed to" |
| **5** | Preview contribution | Expected value, novelty | "This study provides," "Our work advances" |

**Positive Patterns:**
- Open with engaging, field-relevant context
- Cite seminal and recent works appropriately
- Clearly articulate the knowledge gap
- State objectives in measurable, specific terms
- Preview study contribution without overstatement
- End with paragraph outlining paper structure (optional)

### 3. Methods Module

| Subsection | Function | Essential Content | Detail Level |
| :--- | :--- | :--- | :--- |
| **Study Design** | Frame methodological approach | Design type, setting, timeline | Complete specification |
| **Participants/Materials** | Define study subjects | Inclusion/exclusion, sourcing | Full reproducibility detail |
| **Procedures** | Describe implementation | Step-by-step protocols | Sufficient for replication |
| **Measurements** | Specify data collection | Instruments, validity, reliability | Technical specifications |
| **Analysis** | Explain analytical approach | Statistical tests, software, criteria | Complete analytical plan |

**Positive Patterns:**
- Use descriptive subheadings for each methodological component
- Provide manufacturer details for equipment and reagents
- Specify software versions and statistical packages
- Report ethical approval and consent procedures
- Include sample size justification or power calculations
- Reference established protocols with citation and modifications

### 4. Results Module

| Component | Function | Presentation Standard | Integration with Visuals |
| :--- | :--- | :--- | :--- |
| **Participant Flow** | Document sample progression | Numbers at each stage | CONSORT/STROBE flow diagram |
| **Baseline Characteristics** | Establish comparability | Table with demographics | Table reference in text |
| **Primary Outcomes** | Answer main research question | Effect sizes, CI, p-values | Primary figure highlight |
| **Secondary Outcomes** | Provide supporting evidence | Additional metrics | Supplementary figures |
| **Subgroup/Sensitivity** | Demonstrate robustness | Stratified analyses | Forest plots, tables |

**Positive Patterns:**
- Present results in logical order matching methods
- Report exact p-values with confidence intervals
- Reference all figures and tables in text
- Highlight key findings without interpretation
- Include negative or non-significant findings honestly
- Use consistent decimal places and units throughout

### 5. Discussion Module

| Paragraph | Function | Content Focus | Connection to Other Sections |
| :--- | :--- | :--- | :--- |
| **1** | Summarize key findings | Main takeaway without statistics | Echoes Introduction objectives |
| **2-3** | Interpret findings | Mechanisms, explanations | Builds on Results data |
| **4-5** | Compare with literature | Agreements, contradictions | Cites Introduction references |
| **6** | State implications | Theoretical, practical, policy | Extends Introduction significance |
| **7** | Acknowledge limitations | Honest assessment | Informs future research |
| **8** | Future directions | Specific next steps | Creates research continuity |
| **9** | Concluding statement | Final synthesis | Memorable takeaway |

**Positive Patterns:**
- Open with clear summary of main findings
- Provide plausible mechanistic explanations
- Engage meaningfully with prior literature
- Articulate specific implications for practice or policy
- Acknowledge limitations transparently
- End with strong, memorable concluding statement

### 6. Conclusion Module

| Element | Function | Expression Standard | Distinction from Discussion |
| :--- | :--- | :--- | :--- |
| **Core Finding** | Restate main discovery | Concise, no statistics | More condensed than Discussion |
| **Value Statement** | Articulate contribution | Clear impact declaration | Broader than Discussion implications |
| **Final Outlook** | Provide forward look | Actionable recommendation | More visionary than Discussion |

**Positive Patterns:**
- Restate findings without introducing new data
- Connect back to Introduction research question
- State contribution clearly and confidently
- Provide actionable recommendations
- End with memorable, impactful final sentence

## III. Inter-Module Connection and Logical Coherence

### 1. Introduction-Discussion-Conclusion Alignment

| Introduction Element | Discussion Response | Conclusion Reinforcement |
| :--- | :--- | :--- |
| Research question | Direct answer with interpretation | Concise restatement of answer |
| Stated gap | How study fills the gap | Confirmation of contribution |
| Hypothesis | Whether supported or refuted | Final position on hypothesis |
| Previewed contribution | Evidence supporting contribution | Declaration of achieved contribution |
| Significance claim | Validation of significance | Elevated significance statement |

### 2. Methods-Results Correspondence

| Methods Component | Results Component | Alignment Check |
| :--- | :--- | :--- |
| Each analytical method | Corresponding result presented | All methods yield results |
| Each hypothesis tested | Statistical test reported | Complete hypothesis coverage |
| Each measurement described | Data presented | No unmeasured variables |
| Each subgroup defined | Subgroup analysis shown | Complete subgroup coverage |

### 3. Results-Discussion Boundary Maintenance

| Results Section | Discussion Section |
| :--- | :--- |
| Presents *what* was found | Explains *what it means* |
| Reports data and statistics | Interprets patterns and relationships |
| References figures and tables | References prior literature |
| Uses past tense for findings | Uses present tense for interpretations |
| No literature comparison | Extensive literature engagement |
| No interpretation or speculation | Mechanistic and theoretical explanation |

### 4. Cross-Section Transition Strategies

| Transition Type | Example Pattern | Function |
| :--- | :--- | :--- |
| **Introduction → Methods** | "To address this gap, we conducted..." | Signals methodological response |
| **Methods → Results** | "Using this approach, we found..." | Connects procedure to findings |
| **Results → Discussion** | "These findings indicate that..." | Moves from data to meaning |
| **Discussion → Conclusion** | "In conclusion, this study demonstrates..." | Signals final synthesis |
| **Paragraph → Paragraph** | "Furthermore," "In contrast," "Building on this" | Maintains logical flow |

## IV. Academic Language Style and Expression Norms

### 1. Tense Usage Patterns

| Section | Primary Tense | Secondary Tense | Usage Context |
| :--- | :--- | :--- | :--- |
| **Abstract** | Past (findings), Present (conclusions) | N/A | Past for what was done, Present for implications |
| **Introduction** | Present (established knowledge), Past (prior studies) | Present Perfect (ongoing research) | Present for facts, Past for specific studies |
| **Methods** | Past | Present (established protocols) | Past for procedures performed |
| **Results** | Past | Present (figure/table references) | Past for findings, Present for visuals |
| **Discussion** | Past (findings), Present (interpretations) | Future (recommendations) | Mixed based on content type |
| **Conclusion** | Present | Future (directions) | Present for conclusions, Future for outlook |

### 2. Voice Selection Patterns

| Context | Recommended Voice | Example | Rationale |
| :--- | :--- | :--- | :--- |
| **Methods procedures** | Passive (traditional) or Active (modern) | "Samples were collected" / "We collected samples" | Emphasizes procedure or researcher agency |
| **Results findings** | Active (increasingly preferred) | "We observed," "The treatment reduced" | Enhances clarity and ownership |
| **Discussion interpretations** | Active | "We interpret," "Our findings suggest" | Clarifies author position |
| **Established knowledge** | Passive | "It is well known that" | Emphasizes consensus over source |
| **Conclusions** | Active | "We conclude," "This study demonstrates" | Asserts author contribution |

### 3. Formal Expression Patterns

| Informal Expression | Formal Alternative | Context |
| :--- | :--- | :--- |
| "We looked at" | "We examined," "We investigated" | Methods/Results |
| "A lot of" | "Numerous," "Substantial," "Considerable" | All sections |
| "Got" | "Obtained," "Acquired," "Received" | Methods/Results |
| "Showed" | "Demonstrated," "Indicated," "Revealed" | Results/Discussion |
| "Maybe" | "May," "Might," "Could potentially" | Discussion |
| "Really" | "Substantially," "Significantly," "Markedly" | Results/Discussion |
| "Thing" | "Factor," "Element," "Component," "Aspect" | All sections |
| "Big" | "Substantial," "Considerable," "Significant" | All sections |

### 4. Hedging and Certainty Expression

| Certainty Level | Modal Verbs | Adverbs | Example Context |
| :--- | :--- | :--- | :--- |
| **High confidence** | "Demonstrate," "Confirm," "Establish" | "Clearly," "Consistently" | Well-supported findings |
| **Moderate confidence** | "Suggest," "Indicate," "Imply" | "Likely," "Probably" | Interpreted findings |
| **Speculative** | "May," "Might," "Could" | "Potentially," "Possibly" | Hypotheses, mechanisms |
| **Tentative** | "Appear to," "Seem to" | "Perhaps," "Potentially" | Preliminary observations |

## V. Data, Figures, Citations, and Formula Expression Paradigms

### 1. Statistical Reporting Standards

| Element | Standard Format | Example |
| :--- | :--- | :--- |
| **P-values** | Exact values (except p<0.001) | "p=0.032" not "p<0.05" |
| **Confidence Intervals** | 95% CI with brackets | "95% CI [1.23, 4.56]" |
| **Effect Sizes** | With interpretation metric | "Cohen's d=0.65, medium effect" |
| **Means** | With SD or SEM | "Mean=15.3 (SD=2.1)" |
| **Sample Sizes** | Explicit n for each analysis | "n=150 per group" |
| **Percentages** | With numerator/denominator | "45% (67/150)" |

### 2. Figure and Table Reference Patterns

| Reference Type | Standard Format | Example |
| :--- | :--- | :--- |
| **Single figure** | "Figure 1" or "Fig. 1" | "As shown in Figure 1A" |
| **Multiple figures** | "Figures 1-3" or "Figs. 1-3" | "Figures 2 and 3 illustrate" |
| **Table reference** | "Table 1" | "Baseline characteristics (Table 1)" |
| **Supplementary** | "Supplementary Figure S1" | "See Supplementary Table S2" |
| **In-text placement** | Before or after relevant text | "(Figure 1)" or "Figure 1 shows" |

### 3. Citation Integration Patterns

| Citation Position | Format | Example |
| :--- | :--- | :--- |
| **End of sentence** | Number or Author-Year | "...previous work.¹" or "...(Smith et al., 2023)" |
| **Within sentence** | Author as subject | "Smith et al.¹ demonstrated that..." |
| **Multiple citations** | Grouped by relevance | "¹⁻⁵" or "(Smith, 2020; Jones, 2021)" |
| **Sequential reference** | Chronological or importance | Order by publication or relevance |

### 4. Formula and Equation Presentation

| Element | Standard Format | Example |
| :--- | :--- | :--- |
| **Numbering** | Sequential in parentheses | "Equation (1)" or "(1)" |
| **Variables** | Italicized | "*x*, *y*, *z*" |
| **Constants** | Regular font | "sin, cos, log, exp" |
| **Definition** | Immediately after equation | "where *x* represents..." |
| **Reference** | By number in text | "Substituting into Equation (2)..." |

## VI. Reusable Writing Ideas and Narrative Logic

### 1. Introduction Narrative Arc

```
Establish Field Importance
    ↓
Review Key Prior Work
    ↓
Identify Specific Gap
    ↓
State Research Objective
    ↓
Preview Study Contribution
    ↓
Outline Paper Structure (optional)
```

**Reusable Opening Patterns:**
- "[Field] plays a critical role in [application/context]."
- "Recent advances in [area] have enabled [capability]."
- "Understanding [phenomenon] is essential for [outcome]."
- "Despite progress in [area], [gap] remains unresolved."

**Reusable Gap Identification:**
- "However, [specific limitation] has not been addressed."
- "Previous studies have focused on [X], but [Y] remains unclear."
- "Limited evidence exists regarding [specific question]."
- "The mechanism underlying [phenomenon] is not well understood."

**Reusable Objective Statement:**
- "This study aimed to [objective] in [population/context]."
- "We investigated whether [hypothesis] using [approach]."
- "Our objective was to determine [outcome] among [subjects]."
- "We sought to develop/validate/examine [focus]."

### 2. Results Narrative Flow

```
Participant/Subject Flow
    ↓
Baseline Characteristics
    ↓
Primary Outcome (Main Finding)
    ↓
Secondary Outcomes (Supporting Evidence)
    ↓
Subgroup/Sensitivity Analyses (Robustness)
    ↓
Additional/Exploratory Findings (if applicable)
```

**Reusable Result Presentation:**
- "A total of [N] participants were included in the analysis."
- "Baseline characteristics were comparable between groups (Table 1)."
- "The primary outcome occurred in [X]% of the intervention group versus [Y]% of controls."
- "This effect was consistent across prespecified subgroups (Figure 2)."
- "Sensitivity analyses yielded similar results (Supplementary Table S3)."

### 3. Discussion Narrative Structure

```
Summary of Key Findings
    ↓
Interpretation and Mechanism
    ↓
Comparison with Prior Literature
    ↓
Implications (Theoretical/Practical/Policy)
    ↓
Limitations (with Mitigation)
    ↓
Future Research Directions
    ↓
Concluding Statement
```

**Reusable Discussion Openings:**
- "Our study demonstrates that [main finding]."
- "The principal finding of this work is [key discovery]."
- "These results provide evidence that [conclusion]."
- "This study makes three key contributions to [field]."

**Reusable Literature Engagement:**
- "Our findings are consistent with prior work showing [agreement]."
- "In contrast to [author], we observed [difference]."
- "While previous studies focused on [X], our work extends this to [Y]."
- "These results reconcile conflicting reports by demonstrating [resolution]."

**Reusable Implication Statements:**
- "These findings have important implications for [practice/policy/theory]."
- "Our results suggest that [intervention] could be considered for [application]."
- "This work provides a foundation for [future development]."
- "From a [clinical/practical/theoretical] perspective, this indicates [implication]."

**Reusable Limitation Acknowledgment:**
- "Several limitations of this study should be acknowledged."
- "First, [limitation] may have affected [aspect] of our findings."
- "However, [mitigating factor] reduces concern about this limitation."
- "Despite these limitations, our findings remain [robust/valid] because [reason]."

**Reusable Future Direction:**
- "Future studies should investigate [specific question] using [method]."
- "Further research is needed to clarify [uncertain aspect]."
- "Longitudinal studies with [feature] would help establish [relationship]."
- "Replication in [population/setting] would strengthen generalizability."

**Reusable Concluding Statements:**
- "In conclusion, this study provides evidence that [main takeaway]."
- "Taken together, our findings suggest that [broader implication]."
- "This work advances our understanding of [field] by [contribution]."
- "These results establish [finding] as a [significant factor] for [outcome]."

### 4. Paragraph Organization Patterns

| Pattern | Structure | Example Use |
| :--- | :--- | :--- |
| **Topic Sentence + Evidence** | Claim → Supporting data → Interpretation | Results paragraphs |
| **Problem-Solution** | Issue → Approach → Outcome | Methods justification |
| **Compare-Contrast** | Prior work → Current work → Difference | Literature review |
| **General-Specific** | Broad statement → Specific example → Implication | Discussion points |
| **Sequential** | Step 1 → Step 2 → Step 3 → Outcome | Procedural descriptions |

## VII. Top-Tier Paper Characteristics: Rigor, Clarity, Logic

### 1. Rigor Indicators

| Dimension | Positive Pattern | Implementation |
| :--- | :--- | :--- |
| **Methodological Transparency** | Complete reproducibility detail | Full protocols, specifications, versions |
| **Statistical Completeness** | Comprehensive statistical reporting | Effect sizes, CI, exact p-values, power |
| **Ethical Compliance** | Clear ethical documentation | IRB approval, consent, registration |
| **Data Availability** | Open data and code sharing | Repository links, access statements |
| **Limitation Honesty** | Transparent weakness acknowledgment | Dedicated limitations section |
| **Reporting Standards** | Adherence to field guidelines | CONSORT, STROBE, ARRIVE, etc. |

### 2. Clarity Indicators

| Dimension | Positive Pattern | Implementation |
| :--- | :--- | :--- |
| **Sentence Structure** | Clear subject-verb-object order | Avoid nested clauses |
| **Paragraph Unity** | One main idea per paragraph | Clear topic sentences |
| **Terminology Consistency** | Same term for same concept | Defined at first use |
| **Abbreviation Management** | Defined and used consistently | Abbreviation list if extensive |
| **Visual Clarity** | Self-explanatory figures and tables | Complete legends, clear labels |
| **Logical Signposting** | Clear transitions between ideas | "Therefore," "However," "Furthermore" |

### 3. Logic Indicators

| Dimension | Positive Pattern | Implementation |
| :--- | :--- | :--- |
| **Research Question Alignment** | All sections address core question | Introduction-Results-Discussion match |
| **Hypothesis Testing** | Clear hypothesis and test correspondence | Each hypothesis receives analysis |
| **Evidence-Claim Proportionality** | Claims match evidence strength | Appropriate hedging and certainty |
| **Causal Inference Appropriateness** | Causality claims match study design | RCT vs. observational distinction |
| **Alternative Explanation Consideration** | Competing interpretations addressed | Discussion of alternative mechanisms |
| **Conclusion-Evidence Grounding** | All conclusions traceable to results | No unsupported claims |

### 4. Coherence Indicators

| Dimension | Positive Pattern | Implementation |
| :--- | :--- | :--- |
| **Thematic Consistency** | Unified focus throughout | All content serves main objective |
| **Terminological Consistency** | Same terms for same concepts | Glossary for complex papers |
| **Tense Consistency** | Appropriate tense by section | Section-specific tense rules |
| **Voice Consistency** | Consistent active/passive choice | Within-section voice uniformity |
| **Citation Consistency** | Uniform citation style | Journal-specific format |
| **Formatting Consistency** | Standardized headings and structure | Clear hierarchy of sections |

## VIII. Section-Specific Sentence Pattern Library

| Function | Pattern | Example |
| :--- | :--- | :--- |
| **Background** | "[Field] is critical for [application/outcome]." | "Cardiovascular disease is the leading cause of mortality globally." |

## X. Implementation Guide for Writers

### 1. Pre-Writing Preparation

- Define core research question in one clear sentence
- Identify 3-5 key findings that answer the question
- Select target journal and review author guidelines
- Prepare all figures and tables before writing text
- Create outline following IMRAD structure
- Gather all references with complete citation information

### 2. Writing Sequence Recommendation

1. **Methods** (easiest, most factual)
2. **Results** (data-driven, straightforward)
3. **Figures and Tables** (visual organization)
4. **Introduction** (requires framing after results known)
5. **Discussion** (requires results and introduction context)
6. **Conclusion** (final synthesis)
7. **Abstract** (summary of complete paper)
8. **Title** (captures essence of complete work)

### 3. Revision Priorities

| Priority | Focus Area | Key Questions |
| :--- | :--- | :--- |
| **1** | Logical coherence | Does each section flow to the next? |
| **2** | Research question alignment | Does all content serve the main objective? |
| **3** | Evidence-claim matching | Are all claims supported by data? |
| **4** | Clarity and readability | Is language clear and concise? |
| **5** | Formatting and style | Does it meet journal requirements? |
| **6** | References and citations | Are all claims properly supported? |

### 4. Quality Indicators Before Submission

- [ ] Title accurately reflects study content
- [ ] Abstract summarizes all key sections
- [ ] Introduction establishes clear gap and objective
- [ ] Methods enable independent replication
- [ ] Results present all planned analyses
- [ ] Discussion interprets without overstatement
- [ ] Conclusion provides clear takeaway
- [ ] All figures and tables referenced in text
- [ ] All citations complete and accurate
- [ ] Word count within journal limits
- [ ] Formatting matches journal style
- [ ] Language edited for clarity and grammar
